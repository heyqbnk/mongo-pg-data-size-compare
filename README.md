# Compare MongoDB and PostgreSQL data size

This repository is used to find difference in data size taken by MongoDB and PostgreSQL.

Used DBs versions:
- MongoDB - 4.2
- PostgreSQL - 11.5

## Scheme
| Column name | Type |
|---|---|
| `vkUserId` | `integer` |
| `name` | `text` |
| `profileImageUrl` | `text` |
| `isBanned` | `bool` |
| `createdAt` | `datetime` |
